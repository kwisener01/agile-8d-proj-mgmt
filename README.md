# FlowForge - Agile 8D Project Management

A unified platform combining Agile project management with 8D problem-solving methodology.

## Features

- **Agile Board**: Kanban-style board with sprint tracking
- **8D Tracker**: Full 8D problem-solving workflow (D0-D8)
- **Bridge View**: Connect quality defects to agile stories
- **Team Pulse**: Analytics dashboard with velocity and metrics

## Getting Started

### Local Development

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Production Build

```bash
npm run build
npm start
```

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/flowforge)

### Manual Deployment

1. Push your code to GitHub
2. Import your repository in [Vercel](https://vercel.com/new)
3. Vercel will automatically detect Next.js and configure the build settings
4. Click "Deploy"

## Tech Stack

- Next.js 14
- React 18
- Custom styling (no external CSS libraries)
- IBM Plex Mono & Bebas Neue fonts
